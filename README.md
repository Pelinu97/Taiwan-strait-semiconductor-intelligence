# Semiconductor Geopolitics in the Taiwan Strait

**Open-Source Intelligence (OSINT) Analysis of US-China Competition and Implications for Turkey**

---

## 📌 About This Project

This project is an open-source intelligence analysis examining the geopolitical dimensions of semiconductor supply chains in the Taiwan Strait, with a specific focus on TSMC's strategic role and the potential implications for Turkey.

## 🎯 Objectives

- Map the US-China rivalry over semiconductor technology and Taiwan
- Analyze TSMC's position as a critical node in global chip supply chains
- Assess potential risks and opportunities for Turkey in this geopolitical context

## 🗂️ Repository Structure

- `/data` — Open-source datasets and references
- `/analysis` — Intelligence assessments and written reports
- `/visuals` — Maps, charts, and infographics

## 🔍 Methodology

All research is based exclusively on open-source information including academic publications, government reports, think tank analyses, financial disclosures, and news sources.

## ⚠️ Disclaimer

This project is purely academic and based entirely on publicly available information.
